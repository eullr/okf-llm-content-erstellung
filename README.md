# OKF Bundle: LLM-Content-Erstellung

Wissensbundle im [Open Knowledge Format](https://okf.wtf) für die Erstellung und Umstrukturierung von Web-Inhalten, die von generativen Systemen (ChatGPT, Gemini, Perplexity, AI Overviews) extrahiert, zitiert und erwähnt werden sollen.

**Autor:** Eugen Ullrich · [eullrich.com](https://eullrich.com) · hi@eullrich.com
**Version:** 1.2.0 · **Lizenz:** [CC BY 4.0](LICENSE.md)

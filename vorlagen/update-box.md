# Gerüst

```markdown
## Was ist neu?

* MM/JJJJ: [Konkrete inhaltliche Änderung: neue Zahl, neue Quelle,
  geänderte Empfehlung.]
* MM/JJJJ: [Änderung.]
```

# Verwendungsregeln

* Zwei bis vier Punkte, neueste zuerst.
* Nur substanzielle Änderungen eintragen; reine Umformulierungen zählen nicht und riskieren Timestamp-Gaming-Einstufung, siehe [Aktualität](/methoden/aktualitaet.md) und [Anti-Patterns](/referenz/anti-patterns.md).
* Jeder Eintrag korrespondiert mit einer echten Textänderung auf derselben Seite.
